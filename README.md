# ⚖️ VakilSahya

VakilSahya is a full-stack legal assistance platform designed to simplify the creation and management of legal documents. It provides users with an easy-to-use interface to generate documents like loan agreements while leveraging a robust backend for processing and handling requests.

---

## 🚀 Features

* 📄 Generate legal documents (e.g., loan agreements)
* ⚙️ REST API for document processing
* 🌐 Clean and responsive frontend UI
* 📬 API testing support via Postman collection
* ⚡ Fast and scalable architecture

---

## ⚙️ Installation & Setup

### 🔹 Prerequisites

Make sure you have installed:

* Node.js (v16 or above)
* Java (JDK 17 recommended)
* Maven

---

### 🔹 Backend Setup

```bash
cd backend
mvn clean install
mvn spring-boot:run
```

Backend will run on:

```
http://localhost:8080
```

---

### 🔹 Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend will run on:

```
http://localhost:5173
```

---

## 🔗 API Testing

You can test all APIs using the included Postman collection:

* Run requests to test backend endpoints
